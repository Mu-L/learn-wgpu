# Update to Vuepress v2

This update has been a long time coming. For a while I've had difficulties with
vuepress v1 where I couldn't have nested directories for assets as when vuepress
would go to load them it would serve up a 404 page instead. Vuepress v2 fixes
that issue and has more features that I can play around with later.

## Stencil demo working on web!

As part of this update I've gotten the [stencil showcase](../../showcase/stencil/)
working in WASM! Vuepress v2 wasn't strictly required for this, but given
that the showcase examples share a resource directory and that wouldn't work
with how Vuepress v1 handles resources. There were work arounds, but I decided
to use it as an excuse to update to v2.
